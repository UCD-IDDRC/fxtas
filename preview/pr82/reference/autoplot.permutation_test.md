# Plot the distribution of a permuted test statistic

Plot the distribution of a permuted test statistic

## Usage

``` r
# S3 method for class 'permutation_test'
autoplot(object, ...)
```

## Arguments

- object:

  an object of class `"permutation_test"`

- ..., :

  additional arguments (not used)

## Value

a [ggplot2::ggplot](https://ggplot2.tidyverse.org/reference/ggplot.html)
constructing a histogram of the test statistic

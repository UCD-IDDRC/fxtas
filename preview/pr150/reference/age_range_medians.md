# replace decade codes with medians of those decades

replace decade codes with medians of those decades

## Usage

``` r
age_range_medians(x)
```

## Arguments

- x:

  a [character](https://rdrr.io/r/base/character.html) vector

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector corresponding to
`x`

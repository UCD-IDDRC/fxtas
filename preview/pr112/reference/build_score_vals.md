# Construct the table indicating how many levels each biomarker has

Construct the table indicating how many levels each biomarker has

## Usage

``` r
build_score_vals(biomarker_levels)
```

## Arguments

- biomarker_levels:

  a list containing the levels for each biomarker

## Value

a [`matrix()`](https://rdrr.io/r/base/matrix.html) indicating which
levels each biomarker has

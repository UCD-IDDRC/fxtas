# Save run info to file

Save run info to file

## Usage

``` r
save_run_info(output_folder, script = "ordinal-sustain.R")
```

## Arguments

- output_folder:

  a [character](https://rdrr.io/r/base/character.html) indicating where
  to save info

- script:

  [character](https://rdrr.io/r/base/character.html): name of script to
  copy

## Value

[TRUE](https://rdrr.io/r/base/logical.html), invisibly

## Examples

``` r
if (FALSE) { # \dontrun{
save_run_info("output123/")
} # }
```

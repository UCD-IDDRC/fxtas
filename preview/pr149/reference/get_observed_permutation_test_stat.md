# Compute permutation test statistic

Compute permutation test statistic

## Usage

``` r
get_observed_permutation_test_stat(output_folder, dataset_names)
```

## Arguments

- output_folder:

  where to find the relevant pickle files

- dataset_names:

  [`character()`](https://rdrr.io/r/base/character.html) vector of
  dataset name stems

## Value

the observed test statistic (a
[`numeric()`](https://rdrr.io/r/base/numeric.html) scalar)

# Compile biomarker groups table

Compile biomarker groups table

## Usage

``` r
compile_biomarker_groups_table(
  dataset,
  biomarker_group_list = compile_biomarker_group_list(dataset = dataset, ...),
  colors = choose_biomarker_group_colors(biomarker_group_list),
  ...
)
```

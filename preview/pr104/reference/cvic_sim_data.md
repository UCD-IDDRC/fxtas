# CVIC results from simulated data

generated by
[`extract_cvic()`](https://UCD-IDDRC.github.io/fxtas/preview/pr104/reference/extract_cvic.md);
see `SuStaIn-simulated-data.qmd` for the full script to generate this
data

## Usage

``` r
cvic_sim_data
```

## Format

An object of class `CVIC` (inherits from `list`) of length 2.

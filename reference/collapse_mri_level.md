# Collapse levels of a single MRI variable

Collapse levels of a single MRI variable

## Usage

``` r
collapse_mri_level(x, levels = c("Moderate", "Severe"))
```

## Arguments

- x:

  a MRI variable ([factor](https://rdrr.io/r/base/factor.html))

- levels:

  [character](https://rdrr.io/r/base/character.html) string vector of
  levels to collapse

## Value

a modified version of `x`

# data prep function

data prep function

## Usage

``` r
tmp_data_prep(x, show_uncert, biomarker_var = "biomarker_label")
```

## Arguments

- x:

  a `PVD` object (output from
  [`plot_positional_var()`](https://UCD-IDDRC.github.io/fxtas/reference/plot_positional_var.md))

- show_uncert:

  [logical](https://rdrr.io/r/base/logical.html): whether to show the
  uncertainty in the sequential order.

- biomarker_var:

  a [character](https://rdrr.io/r/base/character.html) string indicating
  which column to use

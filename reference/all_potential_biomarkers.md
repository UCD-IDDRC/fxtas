# Get a list of all potential biomarkers available in dataset

Get a list of all potential biomarkers available in dataset

## Usage

``` r
all_potential_biomarkers(dataset)
```

## Arguments

- dataset:

  [data.frame](https://rdrr.io/r/base/data.frame.html) Dataset to
  extract names from

## Value

a [list](https://rdrr.io/r/base/list.html)
